## 🔄 业务流程详图

### 流程1: 选择宽度并锁定物料

```
用户操作                 系统处理                    数据库变化
─────────────────────────────────────────────────────────────
  打开排程详情
      │
      ▼
  点击"选择宽度"
      │                  
      ├──────────────▶  加载可用宽度列表
      │                      │
      │                      ├──────────▶  SELECT width, 
      │                      │             available_area,
      │                      │             available_rolls
      │                      │             FROM film_stock
      │                      │             WHERE available_area > 0
      │                      ▼
      ◀──────────────  显示宽度选项
  宽度下拉框
  [1040mm (1500㎡, 25卷)]
  [1280mm (2000㎡, 30卷)]
      │
      ▼
  选择: 1040mm
      │
      ├──────────────▶  查询该宽度库存详情
      │                      │
      │                      ├──────────▶  SELECT * FROM film_stock
      │                      │             WHERE width = 1040
      │                      ▼
      ◀──────────────  显示库存详情
  库存详情显示:
  ✓ 库存充足
  可用: 1500㎡ (25卷)
      │
      ▼
  输入需求面积: 100㎡
      │
      ▼
  点击"确认锁定"
      │
      ├──────────────▶  执行锁定操作
      │                      │
      │                      ├──1. 查找库存ID
      │                      │
      │                      ├──2. 选择明细卷(FIFO)
      │                      │
      │                      ├──3. 更新film_stock ──▶  UPDATE film_stock
      │                      │                          SET available_area = 1400,
      │                      │                              locked_area = 100,
      │                      │                              available_rolls = 24,
      │                      │                              locked_rolls = 1
      │                      │
      │                      ├──4. 更新明细状态 ──────▶  UPDATE film_stock_detail
      │                      │                          SET status = 'locked'
      │                      │                          WHERE id IN (...)
      │                      │
      │                      ├──5. 创建锁定记录 ──────▶  INSERT INTO 
      │                      │                          schedule_material_lock
      │                      │                          (schedule_id, order_id,
      │                      │                           tape_stock_id, 
      │                      │                           locked_area, ...)
      │                      │
      │                      ├──6. 更新排程表 ────────▶  UPDATE schedule_coating
      │                      │                          SET film_width = 1040,
      │                      │                              film_stock_id = ?,
      │                      │                              material_lock_status = 'locked',
      │                      │                              film_locked_time = NOW()
      │                      ▼
      ◀──────────────  锁定成功
  提示: "物料锁定成功"
      │
      ▼
  表格更新显示:
  膜宽: 1040 (绿色)
  状态: [已锁定] [解锁]
```

### 流程2: 解除物料锁定

```
用户操作                 系统处理                    数据库变化
─────────────────────────────────────────────────────────────
  点击"解锁"按钮
      │
      ▼
  确认解锁
      │
      ├──────────────▶  执行解锁操作
      │                      │
      │                      ├──1. 查询锁定记录 ─────▶  SELECT * FROM
      │                      │                          schedule_material_lock
      │                      │                          WHERE schedule_id = ?
      │                      │
      │                      ├──2. 释放film_stock ───▶  UPDATE film_stock
      │                      │                          SET available_area = 1500,
      │                      │                              locked_area = 0,
      │                      │                              available_rolls = 25,
      │                      │                              locked_rolls = 0
      │                      │
      │                      ├──3. 释放明细 ──────────▶  UPDATE film_stock_detail
      │                      │                          SET status = 'available'
      │                      │                          WHERE id IN (...)
      │                      │
      │                      ├──4. 更新锁定记录 ──────▶  UPDATE 
      │                      │                          schedule_material_lock
      │                      │                          SET lock_status = '已释放',
      │                      │                              released_time = NOW()
      │                      │
      │                      ├──5. 清空排程锁定 ──────▶  UPDATE schedule_coating
      │                      │                          SET film_width = NULL,
      │                      │                              material_lock_status = 'unlocked'
      │                      ▼
      ◀──────────────  解锁成功
  提示: "物料解锁成功"
      │
      ▼
  表格更新显示:
  膜宽: 未选择 (橙色)
  状态: [选择宽度]
```

## 📈 状态转换图

```
物料锁定状态转换:

  [未锁定]  ──选择宽度──▶  [已锁定]  ──领料──▶  [已领料]  ──生产──▶  [已消耗]
unlocked                  locked              allocated           consumed
    ▲                        │                                         
    │                        │                                         
    │                        ▼                                         
    └──────────────────  [已释放]                                     
                         released                                     
                         (取消排程)                                   

图例:
- 未锁定: 初始状态，可以选择宽度
- 已锁定: 物料已锁定，库存扣减，可以解锁
- 已释放: 锁定被释放，库存恢复
- 已领料: 生产已领料，不可解锁
- 已消耗: 生产完成，物料已消耗
```
