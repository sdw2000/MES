### 涂布计算算法
```
7步精确算法：
1. 计算订单总面积 = Σ(宽×长×数量) / 1,000,000
2. 计算涂布长度 = Σ(长×数量) / 1000
3. 确定涂布材料宽度 = MAX(宽度)
4. 计算涂布面积 = 长×宽 / 1,000,000
5. 获取工艺速度 = 参数或默认值(20m/分钟)
6. 计算涂布时间 = 长度 / 速度
7. 创建涂布任务 = 记录所有结果

特点：
- BigDecimal精度控制
- 支持多订单分组
- 详细日志记录
- 异常情况处理
```
