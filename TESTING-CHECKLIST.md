## 🐛 如果遇到问题

### 问题 A: 保存时提示"保存失败"
**检查**：
- 浏览器 Console 是否有错误
- Network 标签中 API 响应的状态码
- 是否所有必填字段都填写了

### 问题 B: 自动计算不正确
**检查**：
- 输入的数字格式是否正确
- 是否填写了所有计算需要的字段（长、宽、卷数、单价）

### 问题 C: 订单号没有自动生成
**检查**：
- 新增时订单号输入框是否为空（应该为空）
- 后端是否正常运行
- 查看后端控制台是否有日志
