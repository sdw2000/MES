### 4. 数据库迁移脚本
创建文件：`src/main/resources/db/migration/alter-customer-fields.sql`

包含以下操作：
1. 创建临时备份字段
2. 从旧字段复制数据到新字段
3. 删除旧字段
4. 创建新字段
5. 从临时字段恢复数据
6. 删除临时字段
7. 修改sales_orders表，添加销售和跟单员字段
8. 从customers表复制销售和跟单员信息到sales_orders表
9. 创建索引提升查询性能

**SQL脚本概要：**
```sql
-- 客户表修改
ALTER TABLE customers CHANGE COLUMN sales_person sales VARCHAR(100);
ALTER TABLE customers CHANGE COLUMN sales_department documentation_person VARCHAR(100);

-- 订单表修改
ALTER TABLE sales_orders ADD COLUMN sales VARCHAR(100);
ALTER TABLE sales_orders ADD COLUMN documentation_person VARCHAR(100);

-- 数据复制
UPDATE sales_orders so
INNER JOIN customers c ON so.customer = c.customer_name
SET so.sales = c.sales, so.documentation_person = c.documentation_person;

-- 创建索引
CREATE INDEX idx_customers_sales ON customers(sales);
CREATE INDEX idx_customers_documentation_person ON customers(documentation_person);
CREATE INDEX idx_sales_orders_sales ON sales_orders(sales);
CREATE INDEX idx_sales_orders_documentation_person ON sales_orders(documentation_person);
```

```

├── sales (VARCHAR 100) → 销售
├── documentation_person (VARCHAR 100) → 跟单员

销售订单表 (sales_orders)
├── sales (VARCHAR 100) → 销售（来自customers）
├── documentation_person (VARCHAR 100) → 跟单员（来自customers）
```
