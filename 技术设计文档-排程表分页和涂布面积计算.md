## Mapper层设计

### 1. 动态SQL模式

所有分页查询Mapper都采用动态SQL模式，使用`<script>`标签和`<if>`条件：

```java
@Select("<script>" +
        "SELECT * FROM table_name " +
        "WHERE 1=1 " +
        "<if test='condition1 != null'>AND col1 = #{condition1}</if>" +
        "<if test='condition2 != null and condition2 != \"\"'>AND col2 LIKE CONCAT('%', #{condition2}, '%')</if>" +
        "ORDER BY col_name ASC " +
        "</script>")
List<Entity> selectPageByCondition(Map<String, Object> params);
```

**优点**：
- 灵活支持多条件查询
- 自动处理NULL条件（不产生无效的WHERE子句）
- 避免SQL注入（使用参数化查询）

### 2. ScheduleOrderItemMapper分页设计

```
selectPageByCondition(Map<String, Object> params)
├─ 参数：scheduleId, status, materialCode
├─ 返回：List<ScheduleOrderItem>
└─ 排序：priority ASC

countByCondition(Map<String, Object> params)
├─ 参数：scheduleId, status
└─ 返回：int（记录总数）
```

**SQL查询**：
```sql
SELECT * FROM schedule_order_item 
WHERE 1=1 
  AND schedule_id = ? 
  AND status = ? 
  AND material_code LIKE ? 
ORDER BY priority ASC
```

### 3. 关联查询设计

对于需要关联设备信息的Mapper（Coating、Rewinding等），使用LEFT JOIN：

```sql
SELECT sc.*, e.equipment_name 
FROM schedule_coating sc 
LEFT JOIN equipment e ON sc.equipment_id = e.id 
WHERE 1=1 
  AND sc.schedule_id = ?
  AND sc.status = ?
ORDER BY sc.plan_start_time ASC
```

**关键点**：
- 使用LEFT JOIN保留即使设备不存在的记录
- 在SELECT中明确指定返回字段
- 表别名清晰（sc表、e表）

## Service层设计

### 1. PageHelper集成模式

```java
@Override
public List<Entity> getEntityList(Map<String, Object> params) {
    // 1. 检查分页参数
    if (params.containsKey("pageNum") && params.containsKey("pageSize")) {
        // 2. 启动分页
        PageHelper.startPage(
            Integer.parseInt(params.get("pageNum").toString()),
            Integer.parseInt(params.get("pageSize").toString())
        );
    }
    // 3. 执行查询（分页信息自动注入）
    return mapper.selectPageByCondition(params);
```

## 涂布面积计算算法

### 1. 算法概述

涂布面积计算是一个7步的多阶段过程：

```
┌─────────────────────────────────────────┐
│ 输入：订单明细列表                      │
│ {id, width, length, rolls, ...}        │
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第1步：计算订单总面积                   │
│ totalOrderSqm = Σ(w×l×qty)/1,000,000   │
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第2步：计算涂布长度                     │
│ coatingLength = Σ(l×qty)/1000           │
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第3步：确定涂布材料宽度                 │
│ jumboWidth = MAX(width)                 │
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第4步：计算涂布面积                     │
│ coatingArea = length × width / 1,000,000│
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第5步：获取工艺速度                     │
│ coatingSpeed = 参数或默认值(20)         │
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第6步：计算涂布时间                     │
│ coatingTime = length / speed             │
└─────────────────┬───────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第7步：创建涂布任务                     │
│ ScheduleCoating.planSqm = coatingArea   │
│ ScheduleCoating.planDuration = time     │
└─────────────────────────────────────────┘
```

### 2. 详细计算步骤

#### 第1步：计算订单总面积

**代码**：
```java
BigDecimal totalOrderSqm = BigDecimal.ZERO;
for (Map<String, Object> item : items) {
    BigDecimal width = new BigDecimal(item.get("width").toString());
    BigDecimal length = new BigDecimal(item.get("length").toString());
    Integer rolls = ((Number)item.get("rolls")).intValue();
    
    BigDecimal itemSqm = width.multiply(length).multiply(new BigDecimal(rolls))
        .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
    
    totalOrderSqm = totalOrderSqm.add(itemSqm);
    
    log.debug("Order sqm: {}mm × {}mm × {} = {} m²", width, length, rolls, itemSqm);
}
```

**示例计算**：
```
订单1：500mm × 1000mm × 10卷 = 5,000,000 mm² = 5 m²
订单2：500mm × 1000mm × 5卷 = 2,500,000 mm² = 2.5 m²
总面积 = 7.5 m²
```

#### 第2步：计算涂布长度

**代码**：
```java
BigDecimal totalLength = BigDecimal.ZERO;
for (Map<String, Object> item : items) {
    BigDecimal length = new BigDecimal(item.get("length").toString());
    Integer rolls = ((Number)item.get("rolls")).intValue();
    
    BigDecimal itemLength = length.multiply(new BigDecimal(rolls));
    totalLength = totalLength.add(itemLength);
}

int coatingLengthMeters = totalLength
    .divide(new BigDecimal(1000), 0, BigDecimal.ROUND_HALF_UP)
    .intValue();

if (coatingLengthMeters < 1) {
    coatingLengthMeters = 1; // 确保最少1米
}
```

**示例计算**：
```
订单1：1000mm × 10 = 10,000mm
订单2：1000mm × 5 = 5,000mm
总长度 = 15,000mm = 15m
```

#### 第3步：确定涂布材料宽度

**代码**：
```java
private Integer determineJumboWidth(List<Map<String, Object>> items) {
    Integer maxWidth = 0;
    
    for (Map<String, Object> item : items) {
        try {
            Integer width = ((Number)item.get("width")).intValue();
            if (width > maxWidth) {
                maxWidth = width;
            }
        } catch (Exception e) {
            log.warn("Error parsing width from item: {}", item);
        }
    }
    
    // 使用最大宽度或默认值
    return maxWidth > 0 ? maxWidth : 1200;
}
```

**优先级**：
1. 订单中最大的宽度
2. 系统默认值1200mm

#### 第4步：计算涂布面积

**代码**：
```java
BigDecimal planSqm = new BigDecimal(coatingLengthMeters)
    .multiply(new BigDecimal(jumboWidth))
    .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);

log.info("Coating area: {}m × {}mm = {} m²", 
         coatingLengthMeters, jumboWidth, planSqm);
```

**示例计算**：
```
涂布面积 = 15m × 500mm / 1,000,000
        = 7,500 / 1,000,000
        = 0.0075 m²
```

**计算精度**：
- 使用`BigDecimal`保证精度
- `divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP)`
  - 第二个参数2：保留2位小数
  - 第三个参数：四舍五入模式

#### 第5步：获取工艺速度

**代码**：
```java
BigDecimal coatingSpeed = new BigDecimal(20); // 默认20m/分钟

Object speedObj = firstItem.get("coating_speed");
if (speedObj != null) {
    try {
        coatingSpeed = new BigDecimal(speedObj.toString());
        if (coatingSpeed.compareTo(BigDecimal.ZERO) <= 0) {
            coatingSpeed = new BigDecimal(20); // 无效值使用默认
        }
    } catch (Exception e) {
        log.warn("Invalid coating speed, using default: {}", speedObj, e);
    }
}
```

**获取顺序**：
1. 订单参数 (coating_speed)
2. 设备参数
3. 系统默认值 (20 m/分钟)

#### 第6步：计算涂布时间

**代码**：
```java
int planDuration;
if (coatingSpeed.compareTo(BigDecimal.ZERO) > 0) {
    planDuration = new BigDecimal(coatingLengthMeters)
        .divide(coatingSpeed, 0, BigDecimal.ROUND_HALF_UP)
        .intValue();
} else {
    planDuration = coatingLengthMeters / 20; // 异常情况处理
}

if (planDuration < 1) {
    planDuration = 1; // 确保最少1分钟
}

log.info("Coating time: {}m / {}m/min = {} minutes", 
         coatingLengthMeters, coatingSpeed, planDuration);
```

**示例计算**：
```
涂布时间 = 15m / 20m/分钟
        = 0.75分钟
        = 1分钟（四舍五入）
```

#### 第7步：创建涂布任务

**代码**：
```java
ScheduleCoating coating = new ScheduleCoating();
coating.setScheduleId(schedule.getId());
coating.setTaskNo(coatingMapper.generateTaskNo());

// 轮询分配设备
Equipment equipment = coatingEquipments.get(equipmentIndex % coatingEquipments.size());
coating.setEquipmentId(equipment.getId());
coating.setEquipmentCode(equipment.getEquipmentCode());
equipmentIndex++;

// 设置计算结果
coating.setPlanLength(new BigDecimal(coatingLengthMeters));
coating.setPlanSqm(planSqm);
coating.setJumboWidth(jumboWidth);
coating.setCoatingSpeed(coatingSpeed);
coating.setPlanDuration(planDuration);

coating.setStatus("pending");
coatingMapper.insert(coating);

log.info("Successfully created coating task: {} (order_area={}m², coating_length={}m, " +
        "coating_area={}m², speed={}m/min, duration={}min)", 
        coating.getTaskNo(), totalOrderSqm, coatingLengthMeters, planSqm, coatingSpeed, planDuration);
```

## 错误处理

### 1. Service层异常处理

```java
@Override
public List<ScheduleOrderItem> getScheduleOrderItems(Map<String, Object> params) {
    try {
        if (params.containsKey("pageNum") && params.containsKey("pageSize")) {
            PageHelper.startPage(
                Integer.parseInt(params.get("pageNum").toString()),
                Integer.parseInt(params.get("pageSize").toString())
            );
        }
        return orderItemMapper.selectPageByCondition(params);
    } catch (NumberFormatException e) {
        log.error("Invalid pagination parameters", e);
        throw new IllegalArgumentException("Invalid pageNum or pageSize");
    } catch (Exception e) {
        log.error("Error fetching schedule order items", e);
        throw new RuntimeException("Failed to fetch schedule order items");
    }
}
```

### 2. Controller层异常处理

```java
@GetMapping("/{scheduleId}/items")
public ResponseResult<Map<String, Object>> getScheduleOrderItems(...) {
    try {
        // 业务逻辑
        List<ScheduleOrderItem> list = scheduleService.getScheduleOrderItems(params);
        PageInfo<ScheduleOrderItem> pageInfo = new PageInfo<>(list);
        
        Map<String, Object> result = new HashMap<>();
        result.put("list", list);
        result.put("total", pageInfo.getTotal());
        
        return ResponseResult.success(result);
    } catch (IllegalArgumentException e) {
        log.error("Invalid parameters", e);
        return ResponseResult.error("Invalid parameters: " + e.getMessage());
    } catch (Exception e) {
        log.error("Error fetching schedule order items", e);
        return ResponseResult.error("Failed to fetch schedule order items");
    }
}
```

### 3. 常见错误场景

| 错误 | 原因 | 解决方案 |
|------|------|--------|
| pageNum/pageSize格式错误 | 参数类型转换失败 | 验证参数类型，提供友好错误提示 |
| scheduleId不存在 | 查询条件不匹配 | 返回空列表而不是抛异常 |
| 数据库连接超时 | 连接池耗尽或网络问题 | 增加连接超时、重试机制 |
| 内存溢出 | 查询结果过大 | 使用分页，限制pageSize |
