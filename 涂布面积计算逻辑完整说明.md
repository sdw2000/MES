# 🎨 涂布面积(planSqm)计算逻辑 - 完整说明

## 📋 概述

涂布计划中的 `planSqm`（计划涂布面积，单位：平方米）是根据以下信息计算得出的：

```
planSqm = 涂布长度(米) × 母卷宽度(毫米) / 1000000
```

---

## 📊 详细计算过程

### 1. 输入数据源

订单信息来自 `SalesOrderItem` 表：

| 字段 | 含义 | 单位 | 示例 |
|------|------|------|------|
| `materialCode` | 产品料号 | - | A001 |
| `width` | 订单宽度 | mm | 1200 |
| `length` | 订单长度 | mm | 1000000 |
| `rolls` | 订单数量 | 卷 | 100 |
| `thickness` | 厚度 | mm | 0.1 |
| `sqm` | 订单面积 | m² | 120 |

> **注意**：length字段在数据库中已从 米(m) 转换为 毫米(mm)
> - 订单输入：1000m
> - 数据库存储：1000000mm

### 2. 涂布长度计算

在 `generateCoatingTasks()` 方法中：

```java
// 第428-432行
Integer coatingLength = 5000; // 暂时使用固定值5000米

coating.setPlanLength(new BigDecimal(coatingLength));
```

**当前实现**：固定值 5000米（待改进）

**理想实现**：应该动态计算
```java
// 动态计算涂布长度：按料号汇总所有订单的总长度
BigDecimal totalLength = BigDecimal.ZERO;
for (Map<String, Object> item : items) {
    BigDecimal length = new BigDecimal(item.get("length").toString());
    Integer rolls = ((Number)item.get("rolls")).intValue();
    
    // 总长度 = sum(单个订单长度)
    totalLength = totalLength.add(length.multiply(new BigDecimal(rolls)));
}

Integer coatingLength = totalLength.intValue() / 1000; // 转换为米
```

### 3. 母卷宽度确定

```java
// 第444-448行
Integer jumboWidth = 1200; // 母卷宽度（mm）

BigDecimal planSqm = new BigDecimal(coatingLength)
    .multiply(new BigDecimal(jumboWidth))
    .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
```

**当前实现**：固定值 1200mm

**改进方案**：
```java
// 方式1：从设备参数获取
Equipment equipment = coatingEquipments.get(equipmentIndex % coatingEquipments.size());
Integer jumboWidth = equipment.getDefaultWidth(); // 从设备配置读取

// 方式2：从订单推断（取最大宽度）
Integer jumboWidth = items.stream()
    .map(item -> ((Number)item.get("width")).intValue())
    .max(Integer::compareTo)
    .orElse(1200);

// 方式3：固定标准
Integer jumboWidth = 1200; // 标准宽度
```

### 4. 平方米计算

```java
BigDecimal planSqm = new BigDecimal(coatingLength)           // 长度(米) = 5000
    .multiply(new BigDecimal(jumboWidth))                     // × 宽度(mm) = 5000 × 1200
    .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
    // ÷ 1000000 = 6.00 平方米
```

**计算公式**：
```
planSqm = (coatingLength × jumboWidth) / 1,000,000
        = (5000 × 1200) / 1,000,000
        = 6,000,000 / 1,000,000
        = 6.00 m²
```

**数学原理**：
- 涂布长度：5000米 = 5000 × 1000 mm = 5,000,000 mm
- 母卷宽度：1200 mm
- 面积 = 长 × 宽 = 5,000,000 mm × 1200 mm = 6,000,000,000 mm²
- 转换为平方米：6,000,000,000 mm² ÷ (1000 × 1000) = 6,000 m²

## ✅ 改进方案

### 方案1：动态计算涂布长度（推荐）

```java
private void generateCoatingTasks(ProductionSchedule schedule, 
                                 Map<String, List<Map<String, Object>>> groupedItems,
                                 String operator) {
    
    for (Map.Entry<String, List<Map<String, Object>>> entry : groupedItems.entrySet()) {
        String materialCode = entry.getKey();
        List<Map<String, Object>> items = entry.getValue();
        
        if (items.isEmpty()) continue;
        
        // ✅ 改进1：计算该料号的总长度
        BigDecimal totalLength = BigDecimal.ZERO;
        Integer maxWidth = 0;
        
        for (Map<String, Object> item : items) {
            // 获取订单长度（毫米）
            Object lengthObj = item.get("length");
            BigDecimal length;
            if (lengthObj instanceof String) {
                length = new BigDecimal((String) lengthObj);
            } else if (lengthObj instanceof Number) {
                length = new BigDecimal(((Number) lengthObj).longValue());
            } else {
                continue;
            }
            
            // 获取订单数量
            Integer rolls = ((Number) item.get("rolls")).intValue();
            
            // 累加总长度
            totalLength = totalLength.add(length.multiply(new BigDecimal(rolls)));
            
            // 记录最大宽度
            Integer width = ((Number) item.get("width")).intValue();
            if (width > maxWidth) {
                maxWidth = width;
            }
        }
        
        // 将毫米转换为米
        int coatingLength = totalLength.divide(new BigDecimal(1000)).intValue();
        
        // ✅ 改进2：使用最大宽度或设备宽度
        Integer jumboWidth = maxWidth > 0 ? maxWidth : 1200; // 优先用订单宽度，否则用默认值
        
        // 创建涂布任务
        ScheduleCoating coating = new ScheduleCoating();
        // ... 其他设置 ...
        
        coating.setPlanLength(new BigDecimal(coatingLength));
        
        // ✅ 改进3：正确计算平方米
        BigDecimal planSqm = new BigDecimal(coatingLength)
            .multiply(new BigDecimal(jumboWidth))
            .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
        
        coating.setPlanSqm(planSqm);
        coating.setJumboWidth(jumboWidth);
    }
}
```

// 需要从以下来源获取：
