# 后端改动点冒烟清单（2026-03-03）

> 目的：10~20分钟快速验证本次后端改动未引入明显回归。
> 基础地址：`http://localhost:8090`
> 鉴权：先登录获取 token，后续请求带 `token` 请求头。

---

## 0. 登录与鉴权

### 0.1 登录
- **POST** `/user/login`
- **Body**:
```json
{ "username": "admin", "password": "123456" }
```
- **期望**：`code=20000`，返回 token。

### 0.2 鉴权接口
- **GET** `/user/info?token=...`
- **期望**：返回当前用户信息与角色。

## 1. 用户管理（本次重点：后端分页）

### 1.1 用户分页列表
- **GET** `/api/users?page=1&size=10&username=`
- **期望**：`code=200/20000`，`data.list` 为数组，且有 `data.total/page/size`。

### 1.2 用户搜索分页
- **GET** `/api/users?page=1&size=10&username=admin`
- **期望**：只返回匹配用户名数据，分页字段仍正确。

### 1.3 新增用户
- **POST** `/api/users`
- **Body**:
```json
{ "username":"smoke_user_01", "password":"123456", "realName":"冒烟用户", "status":0 }
```
- **期望**：创建成功，列表可查到。

### 1.4 编辑用户
- **PUT** `/api/users/{id}`
- **Body**:
```json
{ "realName":"冒烟用户-更新", "status":0 }
```
- **期望**：更新成功，列表值变化。

### 1.5 删除用户
- **DELETE** `/api/users/{id}`
- **期望**：删除成功（`admin` 账号不可删）。
