## 🔍 功能快速查找

### 需要查看「客户优先级评分」？
👉 [CustomerPriorityService.java](src/main/java/com/fine/service/customer/CustomerPriorityService.java)
- 评分公式：`总分 = 新客户20分 + 账期分 + 月均额/30000 + 单价偏差*50`
- 关键方法：`calculatePriorityScore()`

### 需要查看「库存三级匹配」？
👉 [InventoryAllocationService.java](src/main/java/com/fine/service/inventory/InventoryAllocationService.java)
- 匹配顺序：成品 → 复卷 → 母卷
- 关键方法：`allocateInventory()`

### 需要查看「涂布排程」？
👉 [CoatingScheduleService.java](src/main/java/com/fine/service/production/CoatingScheduleService.java)
- 颜色分组：`groupByColor()`
- MOQ检查：`checkMOQAndCreateBatches()`
- 设备分配：`scheduleCoatingBatch()`

### 需要查看「全流程排程」？
👉 [OrderScheduleOrchestrator.java](src/main/java/com/fine/service/orchestrator/OrderScheduleOrchestrator.java)
- 6步流程：优先级 → 库存 → 涂布 → 复卷 → 分切 → 交付
- 关键方法：`batchScheduleOrders()`

### 需要测试系统？
👉 [SchedulingSystemQuickTest.java](src/main/java/com/fine/service/test/SchedulingSystemQuickTest.java)
- 执行：`quickTest.runAllTests()`

## ⚡ 快捷操作

### 验证数据库表
```sql
USE erp;
SELECT TABLE_NAME FROM information_schema.TABLES 
WHERE TABLE_SCHEMA = 'erp' AND TABLE_NAME LIKE '%customer%' OR TABLE_NAME LIKE '%material%';
```

### 刷新统计数据
```java
customerTransactionSummaryMapper.refreshAllStats();
customerMaterialStatsMapper.refreshAllStats();
materialBasePriceMapper.refreshAllStats();
```

### 查看设备日历
```sql
SELECT * FROM equipment_schedule 
WHERE equipment_category = 'coating' 
  AND status = 'scheduled' 
ORDER BY start_time;
```
