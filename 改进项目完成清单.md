## 📝 使用示例

### 示例1：获取排程明细
```bash
curl -X GET "http://localhost:8080/api/production/schedule/123/items?pageNum=1&pageSize=10" \
  -H "Authorization: Bearer token"
```

**响应**：
```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "schedule_id": 123,
        "material_code": "ABC001",
        "width": 500,
        "length": 1000,
        "rolls": 10
      }
    ],
    "total": 100,
    "pageNum": 1,
    "pageSize": 10,
    "pages": 10,
    "hasNextPage": true
  }
}
```

### 示例2：按条件查询涂布任务
```bash
curl -X GET "http://localhost:8080/api/production/schedule/coating/list?scheduleId=123&status=pending&pageNum=1&pageSize=10"
```

### 示例3：按日期和状态查询分切任务
```bash
curl -X GET "http://localhost:8080/api/production/schedule/slitting/list?planDate=2024-01-01&status=pending&pageNum=1"
```
