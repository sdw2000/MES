## ✨ 核心功能速览

### 1. 排程启动时的物料锁定
```
POST /production/schedule-material/lock/{scheduleId}

返回:
{
  "success": true,
  "totalOrders": 5,
  "fullyLockedOrders": 3,
  "partiallyLockedOrders": 1,
  "needCoatingOrderIds": [456]
}
```

### 2. 生产领料
```
POST /production/schedule-material/allocate
Body: { "lockIds": [1, 2, 3] }

效果: 库存自动扣减
```

### 3. 排程取消时的释放
```
POST /production/schedule-material/release/{scheduleId}

效果: 库存自动恢复
```

### 4. 多项查询功能
```
GET /allocation/{scheduleId}           # 查询排程分配
GET /allocation/{scheduleId}/{orderId} # 查询订单分配
GET /tape-locks/{tapeId}               # 查询物料锁定
GET /order-locks/{orderId}             # 查询订单锁定
```

## 💡 关键概念

### 物料三层级
```
优先级 1: 复卷 (成本低，数量有限)
优先级 2: 母卷 (成本中等，数量较多)
优先级 3: 涂布 (成本高，数量无限)
```

### 面积粒度
```
支持按面积(m²)精确锁定
而不是按卷数
更灵活的物料分配
```

### 并发安全
```
使用乐观锁（版本号）
防止并发冲突
自动重试机制
```
