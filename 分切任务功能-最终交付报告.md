## 【关键技术实现】

### 1. 10分钟时间精度处理
```java
private Date roundTimeToTenMinutes(Date date) {
    Calendar cal = Calendar.getInstance();
    cal.setTime(date);
    
    int minute = cal.get(Calendar.MINUTE);
    int roundedMinute = (minute + 5) / 10 * 10;
    
    if (roundedMinute >= 60) {
        cal.add(Calendar.HOUR_OF_DAY, 1);
        roundedMinute = 0;
    }
    
    cal.set(Calendar.MINUTE, roundedMinute);
    cal.set(Calendar.SECOND, 0);
    cal.set(Calendar.MILLISECOND, 0);
    
    return cal.getTime();
}
```

### 2. 自动任务号生成
```java
private String generateSlittingTaskNo() {
    SimpleDateFormat sdf = new SimpleDateFormat("yyyyMMdd");
    String dateStr = sdf.format(new Date());
    
    QueryWrapper<ScheduleSlitting> wrapper = new QueryWrapper<>();
    wrapper.likeRight("task_no", "SLT-" + dateStr);
    long count = slittingMapper.selectCount(wrapper) + 1;
    
    return String.format("SLT-%s-%05d", dateStr, count);
}
```

### 3. 自动规格生成
```java
if (StringUtils.isEmpty(slitting.getSpec())) {
    String spec = String.format("%s-%s x %s x %d卷",
        slitting.getMaterialCode(),
        slitting.getTargetWidth(),
        slitting.getSlitLength(),
        slitting.getPlanRolls() != null ? slitting.getPlanRolls() : 0);
    slitting.setSpec(spec);
}
```

### 4. 多条件分页查询
```java
@Override
public IPage<ScheduleSlitting> getSlittingTasks(Map<String, Object> params) {
    Integer pageNum = (Integer) params.get("pageNum");
    Integer pageSize = (Integer) params.get("pageSize");
    
    Page<ScheduleSlitting> page = new Page<>(pageNum, pageSize);
    QueryWrapper<ScheduleSlitting> wrapper = new QueryWrapper<>();
    
    // 支持多条件组合过滤
    if (!StringUtils.isEmpty(orderNo)) {
        wrapper.like("order_no", orderNo);
    }
    if (!StringUtils.isEmpty(status)) {
        wrapper.eq("status", status);
    }
    if (!StringUtils.isEmpty(planDate)) {
        wrapper.like("plan_start_time", planDate);
    }
    if (equipmentId != null) {
        wrapper.eq("equipment_id", equipmentId);
    }
    
    wrapper.orderByDesc("create_time");
    return slittingMapper.selectPage(page, wrapper);
}
```
