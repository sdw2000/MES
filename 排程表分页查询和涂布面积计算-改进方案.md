### 2. 涂布面积计算逻辑调整
- ❌ 当前计算：固定值不符合实际
- ✅ 改进计算：
  - **订单面积** = 订单宽度 × 订单长度 × 订单数量
  - **涂布面积** = 涂布材料宽度 × 涂布长度
  - **涂布长度**是最终确定的值
  - **涂布时间** = 涂布长度 / 工艺速度（形成时间流）

## 🛠️ 解决方案

### 方案1：修复分页查询问题

#### 问题根因
1. **Mapper中没有分页SQL** - 需要添加分页查询方法
2. **Service没有使用PageHelper** - 分页信息丢失
3. **Controller分页处理不完整** - 分页参数未正确传递

#### 改进步骤

**第1步：在各Mapper中添加分页查询方法**

以`ScheduleOrderItemMapper`为例：

```java
// 添加分页查询方法
@Select("<script>" +
        "SELECT * FROM schedule_order_item " +
        "WHERE schedule_id = #{scheduleId} " +
        "<if test='status != null and status != \"\"'>AND status = #{status}</if>" +
        "ORDER BY priority ASC " +
        "</script>")
List<ScheduleOrderItem> selectByScheduleIdWithPaging(
    @Param("scheduleId") Long scheduleId,
    @Param("status") String status);

// 添加统计查询
@Select("SELECT COUNT(*) FROM schedule_order_item WHERE schedule_id = #{scheduleId}")
int countByScheduleId(@Param("scheduleId") Long scheduleId);
```

**第2步：在Service层使用PageHelper**

```java
public List<ScheduleOrderItem> getScheduleOrderItems(Long scheduleId, Integer pageNum, Integer pageSize, String status) {
    // PageHelper分页必须在执行查询前调用
    PageHelper.startPage(pageNum, pageSize);
    
    // 执行查询（PageHelper会自动拦截）
    return scheduleOrderItemMapper.selectByScheduleIdWithPaging(scheduleId, status);
}
```

**第3步：在Controller中完善分页返回**

```java
@GetMapping("/{scheduleId}/items")
public ResponseResult<Map<String, Object>> getScheduleItems(
        @PathVariable Long scheduleId,
        @RequestParam(defaultValue = "1") Integer pageNum,
        @RequestParam(defaultValue = "10") Integer pageSize,
        @RequestParam(required = false) String status) {
    
    List<ScheduleOrderItem> list = scheduleService.getScheduleOrderItems(
        scheduleId, pageNum, pageSize, status);
    PageInfo<ScheduleOrderItem> pageInfo = new PageInfo<>(list);
    
    Map<String, Object> result = new HashMap<>();
    result.put("list", pageInfo.getList());
    result.put("total", pageInfo.getTotal());
    result.put("pageNum", pageInfo.getPageNum());
    result.put("pageSize", pageInfo.getPageSize());
    result.put("pages", pageInfo.getPages());
    result.put("hasNextPage", pageInfo.isHasNextPage());
    
    return ResponseResult.success(result);
}
```

### 方案2：调整涂布面积计算逻辑

#### 计算公式

```
订单面积(m²) = (订单宽度mm × 订单长度mm × 订单数量) / 1,000,000

涂布面积(m²) = (涂布材料宽度mm × 涂布长度m) / 1,000

涂布时间(分钟) = 涂布长度(m) / 工艺速度(m/分钟)
```

#### 实现代码

在`ProductionScheduleServiceImpl.java`的`generateCoatingTasks()`中改进：

```java
private void generateCoatingTasks(ProductionSchedule schedule, 
                                 Map<String, List<Map<String, Object>>> groupedItems,
                                 String operator) {
    List<Equipment> coatingEquipments = equipmentMapper.selectAvailableByType("COATING");
    
    if (coatingEquipments.isEmpty()) {
        log.warn("No available coating equipment found");
        return;
    }
    
    int equipmentIndex = 0;
    
    for (Map.Entry<String, List<Map<String, Object>>> entry : groupedItems.entrySet()) {
        String materialCode = entry.getKey();
        List<Map<String, Object>> items = entry.getValue();
        
        if (items.isEmpty()) continue;
        
        try {
            Map<String, Object> firstItem = items.get(0);
            
            // ===== 第1步：计算订单总面积 =====
            BigDecimal totalOrderSqm = BigDecimal.ZERO;
            for (Map<String, Object> item : items) {
                // 订单面积 = 宽(mm) × 长(mm) × 数量
                BigDecimal width = new BigDecimal(item.get("width").toString());
                BigDecimal length = new BigDecimal(item.get("length").toString());
                Integer rolls = ((Number)item.get("rolls")).intValue();
                
                BigDecimal itemSqm = width.multiply(length).multiply(new BigDecimal(rolls))
                    .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
                
                totalOrderSqm = totalOrderSqm.add(itemSqm);
                
                log.debug("Order sqm: {}mm × {}mm × {} = {} m²", width, length, rolls, itemSqm);
            }
            
            // ===== 第2步：计算涂布长度 =====
            BigDecimal totalLength = calculateTotalLength(items); // 毫米
            int coatingLengthMeters = totalLength.divide(new BigDecimal(1000), 0, BigDecimal.ROUND_HALF_UP).intValue();
            
            if (coatingLengthMeters < 1) {
                coatingLengthMeters = 1;
            }
            
            log.info("Coating length for material {}: {} meters", materialCode, coatingLengthMeters);
            
            // ===== 第3步：确定涂布材料宽度 =====
            Integer jumboWidth = determineJumboWidth(items);
            
            // ===== 第4步：计算涂布面积 =====
            // 涂布面积 = 涂布材料宽度(mm) × 涂布长度(m) / 1000
            BigDecimal planSqm = new BigDecimal(coatingLengthMeters)
                .multiply(new BigDecimal(jumboWidth))
                .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
            
            log.info("Coating area: {}m × {}mm = {} m²", coatingLengthMeters, jumboWidth, planSqm);
            
            // ===== 第5步：计算涂布时间 =====
            // 获取工艺速度（涂布速度）
            BigDecimal coatingSpeed = new BigDecimal(20); // 默认20m/分钟
            
            // 从工艺库或设备参数获取（可选）
            Object speedObj = firstItem.get("coating_speed");
            if (speedObj != null) {
                try {
                    coatingSpeed = new BigDecimal(speedObj.toString());
                } catch (Exception e) {
                    log.warn("Invalid coating speed, using default");
                }
            }
            
            // 时间 = 长度(m) / 速度(m/分钟)
            int planDuration = coatingLengthMeters / coatingSpeed.intValue();
            if (planDuration < 1) {
                planDuration = 1;
            }
            
            log.info("Coating time: {}m / {}m/min = {} minutes", coatingLengthMeters, coatingSpeed, planDuration);
            
            // ===== 第6步：创建涂布任务 =====
            ScheduleCoating coating = new ScheduleCoating();
            coating.setScheduleId(schedule.getId());
            coating.setTaskNo(coatingMapper.generateTaskNo());
            
            Equipment equipment = coatingEquipments.get(equipmentIndex % coatingEquipments.size());
            coating.setEquipmentId(equipment.getId());
            coating.setEquipmentCode(equipment.getEquipmentCode());
            coating.setEquipmentName(equipment.getEquipmentName());
            equipmentIndex++;
            
            coating.setPlanDate(schedule.getScheduleDate());
            coating.setMaterialCode(materialCode);
            coating.setMaterialName((String)firstItem.get("material_name"));
            coating.setColorCode((String)firstItem.get("color_code"));
            
            if (firstItem.get("thickness") != null) {
                coating.setThickness(new BigDecimal(firstItem.get("thickness").toString()));
            }
            
            if (firstItem.get("order_no") != null) {
                coating.setOrderNo((String)firstItem.get("order_no"));
            }
            
            // 设置计算出的值
            coating.setPlanLength(new BigDecimal(coatingLengthMeters));
            coating.setPlanSqm(planSqm);
            coating.setJumboWidth(jumboWidth);
            coating.setCoatingSpeed(coatingSpeed);
            coating.setOvenTemp(new BigDecimal(120));
            coating.setPlanDuration(planDuration);
            coating.setWorkHours(planDuration);
            
            // 设置订单相关信息
            coating.setOrderSqm(totalOrderSqm); // 订单总面积
            coating.setReleaseLayer("离型纸");
            coating.setFilmWidth(jumboWidth);
            
            coating.setStatus("pending");
            coating.setOrderStatus("pending");
            coating.setCreateBy(operator);
            
            coatingMapper.insert(coating);
            
            log.info("Created coating task: {} (length={}m, area={}m², time={}min)", 
                    coating.getTaskNo(), coatingLengthMeters, planSqm, planDuration);
            
        } catch (Exception e) {
            log.error("Error creating coating task for material: {}", materialCode, e);
            throw new RuntimeException("Failed to create coating task: " + e.getMessage());
        }
    }
}
```

## 📊 时间流逻辑

### 涂布任务时间流

```
涂布长度确定 → 涂布时间计算 → 排程时间分配 → 生产执行

具体步骤：

第1步：确定涂布长度
  input: 订单数据(长度 × 数量)
  formula: 涂布长度(m) = Σ(订单长度 × 订单数量) / 1000
  output: 例如 5000m

第2步：读取工艺速度
  input: 工艺库或设备参数
  typical: 20m/分钟（涂布速度）
  output: 工艺速度参数

第3步：计算涂布时间
  formula: 涂布时间(分钟) = 涂布长度(m) / 工艺速度(m/分钟)
  example: 5000 / 20 = 250 分钟 = 4小时10分钟
  output: 计划工时 = 250分钟

第4步：分配排程时间
  input: 工时 + 班次时间 + 设备状态
  formula: 计划开始时间 + 计划工时 = 计划结束时间
  output: 排程时间窗口确定

第5步：生产执行
  start: 计划开始时间
  during: 按实际进度跟踪
  end: 计划结束时间
  actual: 记录实际时间
```

### 完整的时间流公式

```
涂布时间流 = 涂布长度(m) / 工艺速度(m/分钟)

其中：
  - 涂布长度 = Σ(订单长度mm × 订单数量) / 1000
  - 工艺速度 = 从工艺库查询或设备配置（单位：m/分钟）
  - 结果单位：分钟

示例：
  订单1：1000m × 100卷 = 100,000m
  订单2：2000m × 50卷  = 100,000m
  ─────────────────────────
  总长度：200,000m
  
  工艺速度：20 m/分钟
  
  涂布时间 = 200,000 / 20 = 10,000 分钟 ≈ 166.7 小时 ≈ 7天
```
