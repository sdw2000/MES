### 关键代码片段

**订单自动填充销售和跟单员：**
```javascript
// src/views/sales/orders.vue
onCustomerChange(customerId) {
  const customer = this.customers.find(c => c.id === customerId)
  if (customer) {
    this.editForm.customer = customer.customerCode
    this.editForm.sales = customer.sales || ''
    this.editForm.documentationPerson = customer.documentationPerson || ''
    if (customer.contactAddress) {
      this.editForm.deliveryAddress = customer.contactAddress
    }
  }
}
```
