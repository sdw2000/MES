#### 1.3 ProductionScheduleServiceImpl 实现
**位置**: `e:\java\MES\src\main\java\com\fine\serviceIMPL\production\ProductionScheduleServiceImpl.java`

核心方法实现(约150行新增代码)：

##### getSlittingTasks(Map<String, Object> params)
- 支持动态分页查询
- 支持多条件组合过滤
- 返回分页结果IPage<ScheduleSlitting>

##### addSlittingTask(ScheduleSlitting slitting)
- 自动生成任务号(格式: SLT-YYYYMMDD-XXXXX)
- 自动生成规格字符串(如: FT-001-50mm x 100m x 50卷)
- 应用10分钟时间四舍五入
- 参数校验(订单号必填)

##### updateSlittingTask(ScheduleSlitting slitting)
- 应用10分钟时间四舍五入
- 返回影响行数

##### deleteSlittingTask(Long id)
- 返回影响行数
- 支持逻辑删除

##### 新增工具方法: roundTimeToTenMinutes(Date date)
```java
private Date roundTimeToTenMinutes(Date date)
```
- 将时间四舍五入到最近的10分钟
- 处理60分钟边界(向上进位小时)
- 清空秒和毫秒字段

**样例**:
- 14:23 → 14:20
- 14:26 → 14:30
- 14:55 → 15:00
- 15:59 → 16:00

##### 新增工具方法: generateSlittingTaskNo()
- 按日期生成任务号序列
- 格式: SLT-YYYYMMDD-00001

## 【技术架构图】

```
前端路由(/production-management/slitting)
    ↓
slittingTasks.vue (包装组件)
    ↓
SlittingTasksNew.vue (主业务组件)
    ↓
API: slittingTasks.js
    ├→ /api/production/schedule/slitting/list (GET)
    ├→ /api/production/schedule/slitting/add (POST)
    ├→ /api/production/schedule/slitting/update (PUT)
    ├→ /api/production/schedule/slitting/delete/{id} (DELETE)
    ├→ /api/production/schedule/slitting/start/{id} (POST)
    └→ /api/production/schedule/slitting/complete/{id} (POST)
    ↓
后端 Controller: ProductionScheduleController
    ├→ getSlittingTasks(@RequestParam)
    ├→ addSlittingTask(@RequestBody ScheduleSlitting)
    ├→ updateSlittingTask(@RequestBody ScheduleSlitting)
    ├→ deleteSlittingTask(@PathVariable Long id)
    ├→ startSlittingTask(@PathVariable Long id)
    └→ completeSlittingTask(@PathVariable Long id)
    ↓
业务逻辑: ProductionScheduleServiceImpl
    ├→ getSlittingTasks() - 分页查询
    ├→ addSlittingTask() - 自动生成任务号、规格、应用时间精度
    ├→ updateSlittingTask() - 更新+时间精度处理
    ├→ deleteSlittingTask() - 删除
    └→ roundTimeToTenMinutes() - 时间四舍五入工具
    ↓
数据层: ScheduleSlittingMapper
    ↓
数据库: schedule_slitting 表
```

## 【API 请求示例】

### 查询分切任务列表
```bash
GET /api/production/schedule/slitting/list?pageNum=1&pageSize=20&orderNo=ORD-2024-001&status=pending
```

### 新增分切任务
```bash
POST /api/production/schedule/slitting/add
Content-Type: application/json

{
  "orderNo": "ORD-2024-001",
  "orderDetailNo": "001-1",
  "equipmentId": 1,
  "materialCode": "FT-001",
  "materialName": "红色胶带50mm",
  "targetWidth": 50,
  "slitLength": 100,
  "thickness": 0.5,
  "planRolls": 100,
  "planStartTime": "2024-12-20 14:23:00",
  "planEndTime": "2024-12-20 15:33:00",
  "operatorName": "张三"
}
```
返回后，时间自动调整为:
- planStartTime: 2024-12-20 14:20:00
- planEndTime: 2024-12-20 15:30:00

### 开始任务
```bash
POST /api/production/schedule/slitting/start/123?operator=张三
```

### 完成任务
```bash
POST /api/production/schedule/slitting/complete/123?operator=张三
```
