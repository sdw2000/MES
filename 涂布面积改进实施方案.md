# 🔧 涂布面积(planSqm)改进实施方案

## 📋 概述

本文档提供了改进 `generateCoatingTasks()` 方法的完整实施代码，将涂布长度和母卷宽度从硬编码改为动态计算。

---

## 🎯 改进目标

- ✅ 涂布长度从固定5000米改为动态计算
- ✅ 母卷宽度从固定1200mm改为从订单推断或设备参数获取
- ✅ 保持向后兼容性
- ✅ 添加异常处理和日志记录

### 改进后的 `generateCoatingTasks()` 方法

**文件路径**：`ProductionScheduleServiceImpl.java`  
**当前行号**：408-480

```java
/**
 * 生成涂布任务
 * 
 * @param schedule 排程信息
 * @param groupedItems 按料号分组的订单项（key=materialCode, value=订单列表）
 * @param operator 操作人员
 */
private void generateCoatingTasks(ProductionSchedule schedule, 
                                 Map<String, List<Map<String, Object>>> groupedItems,
                                 String operator) {
    // 获取可用的涂布设备
    List<Equipment> coatingEquipments = equipmentMapper.selectAvailableByType("COATING");
    
    if (coatingEquipments.isEmpty()) {
        log.warn("No available coating equipment found for schedule: {}", schedule.getId());
        return;
    }
    
    int equipmentIndex = 0;
    
    for (Map.Entry<String, List<Map<String, Object>>> entry : groupedItems.entrySet()) {
        String materialCode = entry.getKey();
        List<Map<String, Object>> items = entry.getValue();
        
        if (items.isEmpty()) continue;
        
        try {
            Map<String, Object> firstItem = items.get(0);
            String customerName = (String) firstItem.get("customer");
            
            // ===== 改进1：动态计算涂布长度 =====
            BigDecimal totalLength = calculateTotalLength(items);
            
            // 将毫米转换为米，取整
            int coatingLengthMeters = totalLength.divide(new BigDecimal(1000), 0, BigDecimal.ROUND_HALF_UP).intValue();
            
            // 确保至少1米（防止为0）
            if (coatingLengthMeters < 1) {
                coatingLengthMeters = 1;
            }
            
            log.info("Calculated coating length for material {}: {} meters from {} items", 
                    materialCode, coatingLengthMeters, items.size());
            
            // ===== 改进2：动态确定母卷宽度 =====
            Integer jumboWidth = determineJumboWidth(items);
            
            // 创建涂布任务
            ScheduleCoating coating = new ScheduleCoating();
            coating.setScheduleId(schedule.getId());
            coating.setTaskNo(coatingMapper.generateTaskNo());
            
            // 分配设备（轮询）
            Equipment equipment = coatingEquipments.get(equipmentIndex % coatingEquipments.size());
            coating.setEquipmentId(equipment.getId());
            coating.setEquipmentCode(equipment.getEquipmentCode());
            coating.setEquipmentName(equipment.getEquipmentName());
            equipmentIndex++;
            
            coating.setPlanDate(schedule.getScheduleDate());
            coating.setMaterialCode(materialCode);
            coating.setMaterialName((String) firstItem.get("material_name"));
            coating.setColorCode((String) firstItem.get("color_code"));
            
            if (firstItem.get("thickness") != null) {
                coating.setThickness(new BigDecimal(firstItem.get("thickness").toString()));
            }
            
            // 设置订单相关信息（取第一个订单的信息）
            if (firstItem.get("order_no") != null) {
                coating.setOrderNo((String) firstItem.get("order_no"));
            }
            if (firstItem.get("order_item_id") != null) {
                coating.setOrderItemId(((Number) firstItem.get("order_item_id")).longValue());
            }
            
            // ===== 设置涂布长度和面积 =====
            coating.setPlanLength(new BigDecimal(coatingLengthMeters));
            
            // 计算平方米
            BigDecimal planSqm = new BigDecimal(coatingLengthMeters)
                .multiply(new BigDecimal(jumboWidth))
                .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
            
            coating.setPlanSqm(planSqm);
            coating.setJumboWidth(jumboWidth);
            
            log.info("Coating task: length={} m, width={} mm, area={} m²", 
                    coatingLengthMeters, jumboWidth, planSqm);
            
            // 设置默认工艺参数
            coating.setCoatingSpeed(new BigDecimal(20)); // 20米/分钟
            coating.setOvenTemp(new BigDecimal(120));    // 120℃
            
            // 计算预计时长（分钟）
            int duration = Math.max(coatingLengthMeters / 20, 1);
            coating.setPlanDuration(duration);
            coating.setWorkHours(duration);
            
            // 默认状态为待涂布
            coating.setStatus("pending");
            coating.setOperator(operator);
            coating.setCreatedAt(new Date());
            
            // 保存涂布任务
            coatingMapper.insert(coating);
            
            log.info("Successfully created coating task: {} for material: {}", 
                    coating.getTaskNo(), materialCode);
            
        } catch (Exception e) {
            log.error("Error creating coating task for material: {}", materialCode, e);
            throw new RuntimeException("Failed to create coating task: " + e.getMessage());
        }
    }
}

/**
 * 计算订单的总涂布长度（毫米）
 * 公式：总长度 = sum(订单长度mm × 订单数量)
 * 
 * @param items 订单列表
 * @return 总长度（毫米）
 */
private BigDecimal calculateTotalLength(List<Map<String, Object>> items) {
    BigDecimal totalLength = BigDecimal.ZERO;
    
    for (Map<String, Object> item : items) {
        try {
            // 获取订单长度（毫米）
            Object lengthObj = item.get("length");
            BigDecimal length;
            
            if (lengthObj instanceof String) {
                length = new BigDecimal((String) lengthObj);
            } else if (lengthObj instanceof Number) {
                length = new BigDecimal(((Number) lengthObj).longValue());
            } else if (lengthObj instanceof BigDecimal) {
                length = (BigDecimal) lengthObj;
            } else {
                log.warn("Invalid length type: {} for item: {}", lengthObj.getClass().getName(), item);
                continue;
            }
            
            // 获取订单数量
            Object rollsObj = item.get("rolls");
            Integer rolls;
            
            if (rollsObj instanceof String) {
                rolls = Integer.parseInt((String) rollsObj);
            } else if (rollsObj instanceof Number) {
                rolls = ((Number) rollsObj).intValue();
            } else {
                log.warn("Invalid rolls type: {} for item: {}", rollsObj.getClass().getName(), item);
                rolls = 1;
            }
            
            // 累加：length(mm) × rolls
            BigDecimal itemLength = length.multiply(new BigDecimal(rolls));
            totalLength = totalLength.add(itemLength);
            
            log.debug("Item length calculation: {} mm × {} rolls = {} mm", length, rolls, itemLength);
            
        } catch (Exception e) {
            log.warn("Error calculating length for item: {}", item, e);
            continue;
        }
    }
    
    log.info("Total coating length calculated: {} mm", totalLength);
    return totalLength;
}

/**
 * 确定母卷宽度
 * 
 * 优先级：
 * 1. 使用订单中的最大宽度
 * 2. 使用设备的标准宽度配置
 * 3. 使用系统默认值 1200mm
 * 
 * @param items 订单列表
 * @return 母卷宽度（毫米）
 */
private Integer determineJumboWidth(List<Map<String, Object>> items) {
    Integer maxWidth = 0;
    
    // 第一步：从订单中找最大宽度
    for (Map<String, Object> item : items) {
        try {
            Object widthObj = item.get("width");
            Integer width;
            
            if (widthObj instanceof String) {
                width = Integer.parseInt((String) widthObj);
            } else if (widthObj instanceof Number) {
                width = ((Number) widthObj).intValue();
            } else {
                continue;
            }
            
            if (width > maxWidth) {
                maxWidth = width;
                log.debug("Updated max width to: {} mm", maxWidth);
            }
        } catch (Exception e) {
            log.warn("Error parsing width from item: {}", item, e);
            continue;
        }
    }
    
    // 如果订单中有有效宽度，使用订单宽度
    if (maxWidth > 0) {
        log.info("Using order width as jumbo width: {} mm", maxWidth);
        return maxWidth;
    }
    
    // 第二步：尝试从设备配置获取（如果设备有此属性）
    // 假设Equipment类有getDefaultWidth()方法
    // Integer deviceWidth = equipment.getDefaultWidth();
    // if (deviceWidth != null && deviceWidth > 0) {
    //     log.info("Using device configured width: {} mm", deviceWidth);
    //     return deviceWidth;
    // }
    
    // 第三步：使用系统默认值
    int defaultWidth = 1200;
    log.info("Using default jumbo width: {} mm", defaultWidth);
    return defaultWidth;
}
```

## 📈 计算示例

### 场景：排程3个订单

```
【订单信息】
订单1：长度=1000000mm(1000m), 宽度=1200mm, 数量=100卷
订单2：长度=2000000mm(2000m), 宽度=1200mm, 数量=50卷
订单3：长度=500000mm(500m),   宽度=1000mm, 数量=100卷


【订单1和2同料号(料号A001)】

步骤1：计算总长度
  item1: 1000000 × 100 = 100,000,000 mm
  item2: 2000000 × 50  = 100,000,000 mm
  ─────────────────────────────────
  总计：200,000,000 mm

步骤2：转换为米
  200,000,000 mm ÷ 1000 = 200,000 m

步骤3：确定宽度
  max(1200, 1200) = 1200 mm

步骤4：计算面积
  200,000 m × 1200 mm = 200,000 × 1200 = 240,000,000
  240,000,000 ÷ 1,000,000 = 240 m²

结果：
  ├─ 涂布长度：200,000 米
  ├─ 母卷宽度：1200 mm
  └─ 计划面积：240.00 m² ✓


【订单3（料号A002）】

步骤1：计算总长度
  500000 × 100 = 50,000,000 mm

步骤2：转换为米
  50,000,000 mm ÷ 1000 = 50,000 m

步骤3：确定宽度
  订单宽度 = 1000 mm

步骤4：计算面积
  50,000 m × 1000 mm = 50,000,000
  50,000,000 ÷ 1,000,000 = 50 m²

结果：
  ├─ 涂布长度：50,000 米
  ├─ 母卷宽度：1000 mm
  └─ 计划面积：50.00 m² ✓
```

## 📌 注意事项

1. **单位转换**：
   - 数据库中 `length` 字段存储单位：毫米(mm)
   - 涂布任务中 `planLength` 字段单位：米(m)
   - 计算时需要除以1000进行转换

2. **向后兼容**：
   - 新增方法不会破坏现有代码
   - 可以逐步迁移现有涂布任务

3. **异常处理**：
   - 添加了try-catch块
   - 记录详细的日志信息
   - 异常时回退到默认值

4. **性能考虑**：
   - 为每个料号分组创建一个涂布任务
   - 避免创建过多的小任务
