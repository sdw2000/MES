### 3. 涂布面积计算改进

**文件**：`src/main/java/com/fine/serviceIMPL/production/ProductionScheduleServiceImpl.java`

#### 3.1 计算公式（generateCoatingTasks方法）

**第1步：计算订单总面积**
```
订单面积 = Σ(订单宽度[mm] × 订单长度[mm] × 订单数量) / 1,000,000
单位：m²
```

**第2步：计算涂布长度**
```
涂布长度 = Σ(订单长度[mm] × 订单数量) / 1000
单位：m
```

**第3步：确定涂布材料宽度**
```
涂布材料宽度 = MAX(订单宽度[mm])
优先级：
1. 使用订单中的最大宽度
2. 使用系统默认值1200mm
```

**第4步：计算涂布面积**
```
涂布面积 = 涂布长度[m] × 涂布材料宽度[mm] / 1,000,000
单位：m²
```

**第5步：获取工艺速度**
```
涂布速度 = 从订单或设备参数获取（默认20m/分钟）
单位：m/分钟
```

**第6步：计算涂布时间（形成时间流）**
```
涂布时间 = 涂布长度[m] / 涂布速度[m/分钟]
单位：分钟
```

**第7步：创建涂布任务**
- 记录所有计算结果
- 分配设备（轮询分配可用设备）
- 设置任务状态为pending

#### 3.2 主要改进点

1. **准确的面积计算**
   - 订单面积 ≠ 涂布面积
   - 订单面积是原始订单的面积总和
   - 涂布面积是材料宽度 × 涂布长度的面积

2. **时间流逻辑**
   - 涂布时间 = 涂布长度 / 工艺速度
   - 这个时间用于后续的生产计划和资源分配

3. **日志记录**
   - 详细的DEBUG和INFO日志
   - 便于调试和监控生产过程
   - 记录格式：
     ```
     订单面积、涂布长度、涂布面积、涂布时间等详细信息
     ```

#### 3.3 代码示例

```java
// 计算订单总面积
BigDecimal totalOrderSqm = BigDecimal.ZERO;
for (Map<String, Object> item : items) {
    BigDecimal width = new BigDecimal(item.get("width").toString());
    BigDecimal length = new BigDecimal(item.get("length").toString());
    Integer rolls = ((Number)item.get("rolls")).intValue();
    
    BigDecimal itemSqm = width.multiply(length).multiply(new BigDecimal(rolls))
        .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);
    
    totalOrderSqm = totalOrderSqm.add(itemSqm);
}

// 计算涂布长度
BigDecimal totalLength = calculateTotalLength(items); // 毫米
int coatingLengthMeters = totalLength.divide(new BigDecimal(1000), 0, BigDecimal.ROUND_HALF_UP).intValue();

// 确定涂布材料宽度
Integer jumboWidth = determineJumboWidth(items);

// 计算涂布面积
BigDecimal planSqm = new BigDecimal(coatingLengthMeters)
    .multiply(new BigDecimal(jumboWidth))
    .divide(new BigDecimal(1000000), 2, BigDecimal.ROUND_HALF_UP);

// 获取工艺速度
BigDecimal coatingSpeed = new BigDecimal(20); // 默认

// 计算涂布时间
int planDuration = new BigDecimal(coatingLengthMeters)
    .divide(coatingSpeed, 0, BigDecimal.ROUND_HALF_UP).intValue();
```

2. 验证涂布任务的计算结果：
   - 订单面积 = (500 × 1000 × 10 + 500 × 1000 × 5) / 1,000,000 = 7.5 m²

   - 涂布面积 = 15 × 500 / 1,000,000 = 0.0075 m²
