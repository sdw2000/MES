# 涂布排程薄膜宽度选择和物料锁定功能

## 📋 功能概述

实现了涂布排程时从仓库查询可用薄膜宽度，手动选择宽度，并锁定物料直至实际领料的完整流程。

## ✅ 已完成功能

### 1. 数据库层面
- ✅ 在 `schedule_coating` 表添加薄膜宽度相关字段：
  - `film_width`: 膜宽(mm)
  - `film_stock_id`: 薄膜库存ID
  - `film_locked_time`: 物料锁定时间
  - `material_lock_status`: 物料锁定状态

- ✅ 薄膜库存表 (`film_stock`) 已包含：
  - 宽度、厚度、规格描述
  - 总面积、可用面积、锁定面积
  - 总卷数、可用卷数、锁定卷数

- ✅ 物料锁定表 (`schedule_material_lock`) 用于记录锁定详情

### 2. 后端API
#### 薄膜宽度查询 API
- **GET** `/api/film/available-widths` - 获取可用薄膜宽度列表
  - 参数: `thickness` (可选) - 按厚度筛选
  - 返回: 宽度列表，包含可用面积和卷数

- **GET** `/api/film/stock-detail` - 获取指定规格的库存详情
  - 参数: `width` (必填), `thickness` (可选)
  - 返回: 库存详细信息

- **GET** `/api/film/check-stock` - 检查库存是否充足
  - 参数: `width`, `thickness`, `requiredArea`
  - 返回: 是否有足够库存

#### 物料锁定 API
- **POST** `/api/schedule/material-lock/lock` - 锁定排程物料
  ```json
  {
    "scheduleId": 排程ID,
    "orderId": 订单ID,
    "filmWidth": 薄膜宽度(mm),
    "filmThickness": 薄膜厚度(μm, 可选),
    "requiredArea": 需求面积(㎡),
    "remark": "备注"
  }
  ```

- **POST** `/api/schedule/material-lock/unlock/{scheduleId}` - 释放锁定物料

- **GET** `/api/schedule/material-lock/locked-materials/{scheduleId}` - 获取锁定物料信息

- **POST** `/api/schedule/material-lock/batch-lock` - 批量锁定物料

### 3. 前端界面
#### 涂布计划表格新增字段
- **膜宽(mm)** 列：
  - 已选择：显示绿色宽度数值
  - 未选择：显示橙色"未选择"提示

- **物料锁定** 操作列：
  - 未锁定：显示"选择宽度"按钮（橙色）
  - 已锁定：显示"已锁定"标签 + "解锁"按钮

#### 薄膜宽度选择对话框
**组件**: `FilmWidthSelector.vue`

**功能**:
1. 显示排程基本信息（任务单号、订单号、产品料号、计划面积）
2. 可选输入薄膜厚度进行筛选
3. 下拉选择薄膜宽度：
   - 显示每个宽度的可用面积和卷数
   - 例如: "1040mm (可用: 1500㎡, 25卷)"
4. 输入需求面积（可点击"使用排程面积"自动填充）
5. 实时显示库存充足性：
   - ✓ 库存充足：绿色提示
   - ⚠ 库存不足：橙色警告
6. 输入备注信息
7. 确认锁定（库存不足时按钮禁用）

## 🔄 业务流程

### 场景1: 排程时锁定物料
```
1. 用户创建涂布排程
2. 在排程详情中查看涂布计划表格
3. 点击某行的"选择宽度"按钮
4. 弹出宽度选择对话框
5. 系统自动加载可用薄膜宽度列表
6. 用户选择合适的宽度
7. 系统显示该宽度的库存情况
8. 用户确认需求面积
9. 点击"确认锁定"
10. 系统执行：
    - 在film_stock中锁定对应面积和卷数
    - 在film_stock_detail中锁定具体的卷
    - 在schedule_material_lock中创建锁定记录
    - 更新schedule_coating的film_width和lock_status
11. 表格中该行显示"已锁定"状态
```

### 场景2: 解除物料锁定
```
1. 用户在涂布计划表格中
2. 找到已锁定的排程行
3. 点击"解锁"按钮
4. 确认解锁操作
5. 系统执行：
    - 释放film_stock中的锁定数量
    - 更新film_stock_detail状态为可用
    - 更新schedule_material_lock状态为已释放
    - 清空schedule_coating的锁定信息
6. 表格恢复显示"选择宽度"按钮
```

### 场景3: 实际领料
```
1. 生产开始，仓库进行领料
2. 系统更新schedule_material_lock状态为"已领料"
3. 从锁定库存转为实际消耗
4. 更新film_stock：
    - locked_area减少
    - available_area不变（因为已经减少过）
5. 更新film_stock_detail状态为"已用"
6. 创建film_stock_out出库记录
```

## 📊 数据表关系

```
schedule_coating (涂布排程)
    ↓ film_stock_id
film_stock (薄膜库存总量)
    ↓ id
film_stock_detail (薄膜库存明细/每一卷)
    
schedule_coating
    ↓ schedule_id
schedule_material_lock (锁定记录)
    ↓ tape_stock_id
film_stock_detail
```

## 💡 使用建议

1. **排程前准备**：确保仓库有足够的薄膜库存
2. **宽度选择**：根据产品规格和库存情况选择合适的宽度
3. **及时解锁**：如果排程取消或变更，及时解除物料锁定
4. **库存监控**：定期检查锁定库存，避免过度锁定

## 📌 注意事项

1. 物料锁定后会减少可用库存，但不影响总库存
2. 只有"锁定中"状态的物料可以解锁
3. "已领料"状态的物料不能解锁，需要通过退料流程处理
4. 选择宽度时会自动按先进先出原则分配具体的卷
5. 解锁会按原路径释放库存

## 🔧 后续改进建议

1. **自动推荐宽度**：根据订单规格自动推荐最合适的薄膜宽度
2. **批量操作**：支持批量为多个排程选择宽度和锁定物料
3. **库存预警**：当可用库存低于安全库存时发出预警
4. **锁定超时**：长时间锁定未使用的物料自动释放
5. **锁定历史**：记录物料锁定和释放的完整历史
