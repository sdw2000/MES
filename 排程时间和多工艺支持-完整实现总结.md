## ✨ 核心功能

### 1. 涂布任务时间计算
- **开始时间逻辑**：
  - 如果排程日期是今天且已过8点 → 使用当前时间
  - 否则 → 使用排程日期的 08:00 作为起始时间
- **时长计算**：
  - 涂布速度：30米/分钟
  - 时长 = 总长度(米) / 30
  - 最小时长：10分钟
- **结束时间**：开始时间 + 时长
- **任务间隔**：留10分钟换料准备时间

**代码位置**：[ProductionScheduleServiceImpl.java](e:\java\MES\src\main\java\com\fine\serviceIMPL\production\ProductionScheduleServiceImpl.java#L191-L296)

```java
// 初始开始时间：排程日期的当天8:00开始，或者当前时间（如果排程日期是今天）
Calendar calendar = Calendar.getInstance();
Date scheduleDate = schedule.getScheduleDate();
calendar.setTime(scheduleDate);

// 检查是否是今天
Calendar today = Calendar.getInstance();
boolean isToday = calendar.get(Calendar.YEAR) == today.get(Calendar.YEAR) &&
                 calendar.get(Calendar.DAY_OF_YEAR) == today.get(Calendar.DAY_OF_YEAR);

Date currentStartTime;
if (isToday && today.after(calendar)) {
    // 如果是今天且已经过了8点，使用当前时间
    currentStartTime = new Date();
} else {
    // 否则使用排程日期的8:00作为起始时间
    calendar.set(Calendar.HOUR_OF_DAY, 8);
    calendar.set(Calendar.MINUTE, 0);
    calendar.set(Calendar.SECOND, 0);
    calendar.set(Calendar.MILLISECOND, 0);
    currentStartTime = calendar.getTime();
}

// 设置每个任务的时间
coating.setPlanStartTime(currentStartTime);
cal.setTime(currentStartTime);
cal.add(Calendar.MINUTE, Math.max(duration, 10));
Date endTime = cal.getTime();
coating.setPlanEndTime(endTime);

// 更新下一个任务的开始时间（留10分钟换料时间）
cal.add(Calendar.MINUTE, 10);
currentStartTime = cal.getTime();
```

### 2. 复卷任务生成
- **触发条件**：所有订单默认都需要复卷
- **开始时间**：涂布任务结束时间
- **时长计算**：
  - 复卷速度：50米/分钟
  - 时长 = 卷数 × 每卷长度(米) / 50
  - 最小时长：1分钟
- **任务间隔**：留5分钟准备时间

## 🔧 调用逻辑

### autoSchedule方法改进
**代码位置**：[ProductionScheduleServiceImpl.java](e:\java\MES\src\main\java\com\fine\serviceIMPL\production\ProductionScheduleServiceImpl.java#L1089-L1136)

```java
// 生成涂布任务（按料号分组）
Date nextStartTime = null;
if (!orderItems.isEmpty()) {
    // 1. 生成涂布任务
    Map<String, List<Map<String, Object>>> groupedItems = orderItems.stream()
        .collect(Collectors.groupingBy(item -> 
            item.get("material_code").toString()
        ));
    
    nextStartTime = generateCoatingTasks(schedule, groupedItems, operator);
    
    // 2. 生成复卷任务
    List<Map<String, Object>> rewindingItems = orderItems.stream()
        .filter(item -> true) // 所有订单都需要复卷
        .collect(Collectors.toList());
    
    if (!rewindingItems.isEmpty()) {
        nextStartTime = generateRewindingTasks(schedule, rewindingItems, operator, nextStartTime);
    }
    
    // 3. 生成分切任务
    List<Map<String, Object>> slittingItems = orderItems.stream()
        .filter(item -> true) // 所有订单都需要分切
        .collect(Collectors.toList());
    
    if (!slittingItems.isEmpty()) {
        nextStartTime = generateSlittingTasks(schedule, slittingItems, operator, nextStartTime);
    }
}
```
